## ОБЯЗАТЕЛЬНЫЕ ИЗОБРАЖЕНИЯ

### 1. Логотип

| Файл | Формат | Размер | Содержание |
|------|--------|--------|------------|
| `logo.svg` | SVG | — | Символ: стилизованная буква V в форме сейфа или кристалла. Текст «Vaultex» рядом шрифтом Sora Bold. Цвет: белый текст + синий акцент #3D7EFF. Нет теней, нет 3D-эффектов. Должен читаться на тёмном фоне. |
| `logo-mark.svg` | SVG | 40×40px | Только символ (без текста). Используется в favicon и мобильных иконках. |
| `logo-dark.svg` | SVG | — | Тёмная версия логотипа для светлых фонов (для OG-изображения) |

**Что должно быть на логотипе:**
- Минималистичная геометрическая форма: щит с ключом внутри, или сейф-ячейка, или кристалл
- Цвет основного символа: градиент от #3D7EFF до #60A5FA (синий)
- Акцентная точка или линия: золотой #F0B429
- Без рамок, теней, скруглений на самом символе

## ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ КО ВСЕМ ИЗОБРАЖЕНИЯМ

- Формат для фотографий/фонов: **WebP** (с fallback JPG)
- Формат для иллюстраций с прозрачностью: **PNG**
- Для всего масштабируемого: **SVG**
- Оптимизация: все PNG/JPG сжать через squoosh.app или TinyPNG (max quality при min size)
- Ретина: для hero и ключевых иллюстраций предусмотреть `@2x`
- Alt-тексты: прописаны в БД (таблица banners) и в коде для статики
- Ленивая загрузка: `loading="lazy"` для всего, что ниже первого экрана
